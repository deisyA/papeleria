Descripción general

Este aplicativo es de consulta interna y contiene la información de los productos que se hayan ingresado previamente. Pretende informar y actualizar el stock de productos, además de mostrar parámetros como precio y descripciones según el producto que se desee visualizar.

Contiene una página de aterrizaje con información de contacto y accesos directos a las plataformas más usadas por el cliente, incluyendo redes sociales.

Algunas funcionalidades

Gestión de usuarios

Permite el registro de usuarios del sistema mediante un formulario de registro.
Guarda sus contraseñas encriptadas.
Retira credenciales de usuario rápidamente.

Repositorio de información

Almacena información de empleados
Fechas y formas de documentos
Inventario de productos
